# 🤝 Contributing to InkHaven

First off, thank you for considering contributing to InkHaven! It's people like you that make InkHaven such a great tool.

### **Getting Involved**

As a proprietary project, we maintain tight control over the core features, but we welcome community contributions in the following areas:
- **🐛 Bug Reports**: Found a glitch? Let us know with a detailed description.
- **💡 Feature Ideas**: Think we're missing a mood or a cool AI suggestion? We'd love to hear it.
- **🎨 UI/UX Feedback**: If you have thoughts on how to make the haven more beautiful.

### **Submission Guidelines**

1.  **Fork the repository** to your personal GitHub account.
2.  **Create a new branch** for your feature or bug fix: `git checkout -b feature/your-feature-name`.
3.  **Ensure code quality**: Run `npm run lint` and verify your changes don't break existing functionality.
4.  **Submit a Pull Request**: Describe your changes in detail and link any related issues.

### **Code of Conduct**

InkHaven is a space for safe, respectful, and anonymous connection. We expect all contributors to adhere to our standards of professional and empathetic conduct.
